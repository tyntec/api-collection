# Silent Authentication Services

In this section you find documents related to tyntec's Silent Authentication services.

The Silent Authentication API provides OAuth2-based authentication that verifies whether an authentication
request originates from the same subscriber by resolving network session information (RADIUS/CGNAT) and
returning a `mobile_id` claim in the ID token when verification succeeds.

The most recent version can be found as well on [Silent Authentication API](https://api.tyntec.com/reference/silent-authentication/current.html)
